Pipeline for analysis of Puccinia striiformis f.sp. tritici genomic reads

## Running the tests

Run the tests:

```bash
./test/run_tests.sh
```

Run the tests and report code coverage:

```bash
./test/run_tests.sh COVERAGE
```

Only run the unit tests:

```bash
./test/run_tests.sh SKIP_INTEGRATION
```
